📌 Instructions:

1. Create a folder named "data" in your Arduino sketch folder

2. Place all web files (HTML, CSS, JS, images) inside the "data" folder

3. Use "ESP32 Sketch Data Upload" tool to create a LittleFS .bin file

4. Select the .bin file below and click UPDATE
